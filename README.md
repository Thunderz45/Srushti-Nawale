# Srushti Nawale - Premium Influencer Portfolio

A modern, elegant, and highly interactive digital media kit and portfolio website for **Srushti Nawale**, a lifestyle, fashion, and travel content creator based in Pune, India.

## Features
- **Luxury Glassmorphism Aesthetic**: Beautiful rose gold, soft lavender, and light pink design.
- **Real Image Showcase**: High-quality visual cards.
- **Campaign Reels Integration**: Autoplay campaign videos with sound controllers.
- **Story Highlights Modal**: Cycle highlight reels with animated progress bars.
- **Sleek Light/Dark Mode**: Persistable theme switcher.
- **100% Mobile Responsive**: Fully optimized layout for all devices.
